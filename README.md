# Dotfiles

## Summary
Upgrade your coding experience with these dotfiles

### Current Setup
1. Bash
2. Vim
3. Git
4. brew
5. ack
6. git_completion OR bash_completion

## Installation (Mac vs Raspberry Pi)
```
cd ~
git clone https://github.com/jchen187/Dotfiles.git
cd Dotfiles

// Mac
make all

// Raspberry Pi
make raspberry
```
